# Statistical Inference and Hypothesis Testing Using Python

## Overview

This project implements classical statistical inference techniques using Python.

The project performs:

- Grouped random sampling
- Population mean estimation
- Population variance estimation
- Confidence interval estimation
- One-sample t-test
- Chi-square hypothesis testing

## Technologies Used

- Python
- NumPy
- SciPy

## Statistical Concepts

- Random Sampling
- Sample Mean
- Sample Variance
- Confidence Intervals
- Hypothesis Testing
- t-Test
- Chi-Square Test

## Applications

- Biostatistics
- Data Analysis
- Biomedical Research
- Statistical Inference
